Elements of the triangle:
- $L_0$ - base
- $h$ - height perpendicular to the base
- $L_1, A_1$ - first side adjacent to the base and the angle opposing that side
- $L_2, A_2$ - second side adjacent to the base and the angle opposing that side
- $A_0$ - angle opposing the base
